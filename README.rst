Python Jawbone API wrapper.


* Free software: MIT license
* Documentation: https://python-jawbone.readthedocs.io.


Features
--------

* Body events
* Band events
* Heart rates
* Custom events
* Goals
* Meals
* Mood
* Moves
* Sleeps
* Timezone
* Trends
* User information
* Friends
* Workouts
